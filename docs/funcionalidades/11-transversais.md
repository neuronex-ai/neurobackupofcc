# Funcionalidades Transversais — O Que Funciona em Todo o Sistema

> Busca universal, modo escuro, app desktop, segurança, tempo real — funcionalidades que permeiam toda a experiência.

---

## O que são

Existem funcionalidades que não pertencem a uma aba específica — elas estão em todo lugar. São os pilares que sustentam a experiência premium do NeuroNex: da busca universal ao modo escuro, do app desktop à criptografia.

---

## Funcionalidades

### 1. Command Menu (⌘K)
Busca universal do sistema. Pressione ⌘K (ou Ctrl+K) e encontre qualquer coisa: pacientes, notas, agendamentos, configurações. É o jeito mais rápido de navegar.

### 2. Navegação Inteligente
Barra de navegação superior com acesso direto a todas as abas do sistema. Design fluido que se adapta ao conteúdo.

### 3. Responsividade Mobile
Cada página tem uma versão mobile otimizada. Não é "desktop comprimido" — são telas redesenhadas para toque e busca rápida.

### 4. Tour Guiado (Onboarding)
Tour interativo para novos usuários. Passo a passo visual que apresenta cada funcionalidade do sistema de forma intuitiva.

### 5. App Desktop (Electron)
Versão nativa para Windows e Mac. Barra de título personalizada, atualizações automáticas, performance superior.

### 6. Dark/Light Mode
Dois temas: **Liquid Glass** (dark) e **Ceramic** (light). Transição suave entre eles, sem piscada na tela.

### 7. Criptografia & Segurança
Dados criptografados, Row Level Security (RLS) no banco de dados, PIN financeiro para operações sensíveis. Proteção de nível bancário.

### 8. Tempo Real (Realtime)
Agendamentos, dashboard e notificações atualizados em tempo real. Se algo muda no sistema, você vê instantaneamente.

### 9. Controle por Plano (Feature Gate)
Funcionalidades protegidas por plano de assinatura. O sistema sabe o que está disponível e guia para upgrade quando necessário.

### 10. Confirmação Externa de Agendamento
Link público que o paciente recebe para confirmar sua sessão com um clique.

### 11. Anamnese Externa
Formulário de anamnese que o paciente preenche fora do sistema — sem precisar de login. As respostas aparecem automaticamente no prontuário.

### 12. Cookie Consent (LGPD)
Banner de consentimento de cookies na landing page, em conformidade com a LGPD.
